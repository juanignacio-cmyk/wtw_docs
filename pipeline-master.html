1.2 — What's Wrong: The Diagnosis

Structural Problems

  • SQL and Prospecting are MERGED — mixing "found a name" with "qualified to sell"
  • No enrichment stage — leads enter raw, never get enriched
  • No Discovery stage — team jumps to demo without understanding pain
  • No Propose stage — negotiation starts without a formal proposal
  • Lifecycle and Pipeline are MIXED — SQL is a lifecycle stage used as pipeline
  • No standardized entry criteria — anyone can move deals anywhere
  • "Opportunity" is vague — what does it actually mean?

Process Problems

  • No enrichment process — SDR receives lead, does nothing with data
  • No clear handoff between SDR and AE
  • No "next action" requirement per deal (Truth System violation)
  • Team tries to "generate the problem" in Negotiation instead of Discovery
  • No time limits per stage — deals rot forever
  • No automation or alerts for stuck deals

SDR

  5. A11Y Maturity estimate: Advanced / Committed / Aware / Unaware (from public info)
  6. Budget signals: Tax funded? Membership? Government grant? Tourism levy?
  7. Document in HubSpot: All properties filled before moving to Outreach

📞 Outreach — Multi-Channel Activation

Pre-Pipeline
🎯 Objective
Activation — Get a response and book a meeting via multiple channels
👤 Owner
SDR
⏱️ Time Limit
21 days max (full sequence, 3-6 touches)
📊 Key Metric
Reply rate, meeting booked rate, Outreach → SQL conversion (target: 20%)
✅ Entry Criteria
First outbound touch sent (any channel). Enrichment complete from Prospecting.
🚪 Exit Criteria
  • Success → SQL Gate: Positive reply + passes 4 qualification checks
  • Pause → Circle Back: 21d, 6 touches, no response (but org is ICP fit)
  • Fail → Closed Lost: Explicit no / bounced / anti-ICP
🔧 Tools/Channels (Multi-Channel — Not Just Email!)
  • Email sequence: 3-5 personalized emails over 2-3 weeks (reference THEIR pain, not WTW features)
  • LinkedIn: Connection request + InMail + comment on their posts
  • Phone calls: Direct dial after warm email touch
  • Event follow-ups: Post-conference personalized outreach within 48hr
  • Referral introductions: Warm intros from existing customers or partners
  • Video messages: Loom/Vidyard personalized 60-sec clips for Tier 1
Multi-Channel Sequence (example for Tier 1):
Day 1: Email #1 (pain-based) + LinkedIn connect → Day 3: LinkedIn message → Day 5: Phone call → Day 8: Email #2 (case study) → Day 12: Video message → Day 16: Email #3 (event/urgency) → Day 21: Final touch or Circle Back

Bad Email "Somos Wheel the World, hacemos verificación de accesibilidad. ¿Quieres una demo?"

⛔ SQL — Sales Qualified Lead

QUALIFICATION GATE — Not a Stage
SQL is a GATE, not a stage
Deals don't "sit" in SQL. They pass through it. If they can't pass, they stay in Outreach or go to Circle Back.
🎯 Objective
Qualification — Confirm 4 criteria before pipeline entry. SDR → AE handoff.
👤 Owner
SDR (qualifies) → AE (takes ownership)
⏱️ Time Limit
Max 3 days as a checkpoint. Not a holding stage.
📊 Key Metric
SQL pass rate, SQL → Discovery conversion (target: 90%)
✅ Entry Criteria (ALL 4 must be YES)
  • Authority: Talking to decision maker or strong champion
  • Need: Identified a real problem or initiative
  • Budget: Can they pay? (>$20K budget, funding identified)
  • Timeline: Is there urgency? (fiscal year, event, mandate)
🚪 Exit Criteria
  • Pass → Discovery: All 4 checks YES, AE takes ownership, Discovery call scheduled
  • Fail → Outreach: 1 criteria missing — work to resolve, max 14 days
  • Fail → Circle Back: 2+ criteria missing but org is ICP
  • Fail → Closed Lost: Not ICP after deeper review
🔧 Tools
HubSpot (4 checkbox properties enforced), Internal handoff meeting/Slack, Discovery call scheduling
Example "The director of Visit Chicago said they have $50K allocated for accessibility this year and wants to see a demo before March." → Authority ✓ Need ✓ Budget ✓ Timeline ✓ → SQL PASS → Enter Pipeline

🔎 Discovery

Pipeline Stage 1
🎯 Objective
Understanding — Gap Selling call. Listen, don't pitch. Understand the problem before showing anything.
👤 Owner
AE (leads the call)
⏱️ Time Limit
Call within 7 days, summary within 24hrs post-call. Max 14 days in stage.
📊 Key Metric
Gap Canvas completion rate, Discovery → Demo conversion (target: 70%)
✅ Entry Criteria
SQL gate passed. Discovery call date set. Decision maker or champion confirmed.
🚪 Exit Criteria
  • Success → Demo: GAP identified, scope clear, AE says "Continue"
  • Pause → Circle Back: GAP not urgent / timing wrong
  • Fail → Closed Lost: No GAP found, not a fit
🔧 Tools
Zoom/Google Meet, Gap Selling Canvas template, HubSpot notes, Discovery Summary Doc (internal), CRM data from enrichment

DO NOT SELL. LISTEN. This is where you "generate the problem" — by helping them see what they don't see.

Gap Selling Questions:

  • "¿Cuántos viajeros con discapacidad reciben al año?" (probably don't know → gap)
  • "¿Cómo miden el impacto económico del turismo accesible?" (don't measure → gap)
  • "¿Qué pasa si un viajero con silla de ruedas llega y el hotel no está preparado?" (reputation risk)
  • "¿Sus competidores están haciendo algo en accesibilidad?" (FOMO)

Identify A11Y Maturity: None → Aware → Active → Advanced

CRITICAL: If you reach Demo without completing Discovery, you'll waste everyone's time with a generic presentation.

🎯 Demo

Pipeline Stage 2
🎯 Objective
Presentation — Show the solution custom to THEIR pain points from Discovery. Be a consultant, not a slide reader.
👤 Owner
AE
⏱️ Time Limit
14 days post-discovery. Proposal in 5 days post-demo.
📊 Key Metric
Demo → Propose conversion (target: 60%), demo show rate
✅ Entry Criteria
Discovery Completed + Gap Canvas done + AE Decision = "Continue"
🚪 Exit Criteria
  • Success → Propose: Client interested, quote being prepared
  • Back → Discovery: Demo cancelled, needs more discovery
  • Pause → Circle Back: "Interesting but not now"
  • Fail → Closed Lost: Not interested after seeing solution
🔧 Tools
Zoom/Meet, Custom demo deck (built from Discovery), Economic Impact Dashboard, Case study library, Meeting recap template
  1. Custom demo based on discovery insights
  2. Show similar DMO case studies (same tier, same region)
  3. Show Economic Impact Dashboard with their market data
  4. End with: "Does this solve what we discussed?"
  5. Send meeting recap within 24h
Example — Nashville "Destinations como Grand Rapids aumentaron 23% en reservas accesibles. Nashville tiene 150+ hoteles y 0 verificados. El mercado potencial es $12M/año."

Pipeline Stage 3
🎯 Objective
Formalization — Send formal proposal with scope, pricing, ROI. Quality-gated (score ≥24/30).
👤 Owner
AE
⏱️ Time Limit
Response in 14 days, max 30 days in stage
📊 Key Metric
Proposal Quality Score (min 24/30), Propose → Negotiation conversion (target: 50%)
✅ Entry Criteria
Amount > $0, Close Date set, Demo completed
🚪 Exit Criteria
  • Success → Negotiation: Client engages — questions, counter-offer
  • Pause → Circle Back: "Let me think about it" / no response 30d
  • Fail → Closed Lost: Explicit rejection
🔧 Tools
Proposal template (Gap-based), PandaDoc/Google Docs, ROI calculator, Case study library, HubSpot Quotes
  1. Custom proposal within 48h of demo
  2. Include specific ROI projection for their destination
  3. Include scope: number of listings, timeline, deliverables
  4. Include case study ROI from similar DMO
  5. Send and schedule follow-up call in 5 days

💰 Negotiation

Pipeline Stage 4
🎯 Objective
Terms — Align pricing, scope, and timeline. Protect value. Close Date MANDATORY.
👤 Owner
AE (Head of Growth joins for Tier 1 deals)
⏱️ Time Limit
Max 45 days, weekly check-ins mandatory
📊 Key Metric
Negotiation → Won conversion (target: 60%), avg discount %, days in stage
✅ Entry Criteria
Close Date MANDATORY, Amount > $0, Proposal sent
🚪 Exit Criteria
  • Success → Closing Soon: Verbal yes / PO received
  • Pause → Circle Back: "Not yet" / stalled
  • Fail → Closed Lost: "No" / no activity 30d past close date
🔧 Tools
HubSpot deal tracking, Negotiation playbook, Discount approval matrix, Weekly pipeline review
  1. Handle objections using Gap Selling (return to pain, don't discount)
  2. If "too expensive" → revisit Discovery pain, show ROI
  3. Weekly touch-base — never go 7+ days without contact
  4. Document every conversation — facts, not intentions
  5. Identify blockers: legal, budget approval, champion change

4.3 — Weekly Bottleneck Review

The "Odd Number" Method (30X)

Each week, compare conversion rates across stages. The stage that deviates most from target is the problem of the week. Focus all coaching and process improvement there until it normalizes.

Example: If Outreach→SQL drops to 10% (target 20%), the weekly focus is: review outreach sequences, check ICP targeting, shadow SDR calls, audit multi-channel mix.

Organization Type Handling

Different org types, different playbooks — same pipeline structure

Use Organization Type (company property) to filter — NOT a deal property.

5.1 — DMO (Direct)

Standard pipeline as described above

  • Standard sales cycle (60-90 days target)
  • Decision maker: Director/VP of Marketing or Partnerships
  • Budget: Marketing or economic development funds
  • Value: Single contract, $20K-$100K ACV
  • ICP Tiers: 1 (top metros), 2 (mid-size), 3 (small/niche)

5.2 — Agency / Partnership

Miles, MMGY, Simpleview — longer cycle, higher value

  • Longer cycle: 6-12 months
  • Different qualification: VP Partnerships/BD
  • Value = multiplier (1 agency = 50+ DMOs)
  • Same pipeline stages, different playbook per stage
  • Requires executive alignment (Head of Growth involvement)
